# minimal-hypr
Minimal config hyprland for arch based distro

Copy hypr folder into: 
```sh 
~/.config/hypr
```
and waybar folder into 

```sh 
~/.config/waybar
```

## For hypaper 

go to hypaper.conf, set the default variable asked (CHANGEME default value)
